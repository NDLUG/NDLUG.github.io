Raspberry Pi Showcase

2 minute read Published: 19 Nov, 2019

LUG members show off Raspberry Pi projects

LUG members show off some Raspberry Pi projects.

Retro Pi


LUG members battle it out in Super Mario Kart via a SNES emulator running on a Raspberry Pi.

One of the projects brought to this week’s meeting was a Raspberry Pi running RetroPie. RetroPie is a project that runs on top of Raspbian, a linux distribution made for Raspberry Pi’s based on Debian Linux. It can be installed on an existing Raspbian installation.


RetroPie allows you to turn your Raspberry Pi into a classic game emulator. It can emulate NES, SNES, Gameboy Advance, and more. All you have to do is download ROMs of the games you want to play to the appropriate folder on the Raspberry Pi, and voila! - you can fulfill that nostalgic desire to play Super Mario Bros 4.

Samba Server


Noah shows off some of the capabilities of a Samba server to other members of LUG.

Another project on display at this week’s meeting was a Samba Server. A Samba Server is an open-source server with its own protocol that allows it to communicate with Windows devices as a native application. It offers services for files and printing, among other things, which is the service we used in conjunction with Kodi.

Kodi Media Center


Logan shows the configuration settings in Kodi.

Kodi is an open source project for streaming media. It can be used in conjunction with a Samba Server to stream media. Basically, the Samba Server hosts and serves files, such as movies. Then, in Kodi, one can configure the application to use a shared folder on the Samba Server, giving access to the server’s movies, pictures, etc. This seems to be a great alternative to a Roku or even Chromecast.


After getting things set up and working, some LUG members started to watch Moana via Kodi!


Thanks to everyone who brought a project in this week, and thanks to those who came to the showcase! This Sunday will be LUGsgiving, we hope to see a good turnout!
